# Blog

Personal blog built with SvelteKit and MDsveX.

## Tech Stack

- **SvelteKit** - Web framework
- **MDsveX** - Markdown for Svelte components
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling

## Development

Install dependencies:

```sh
bun install
```

Start the development server:

```sh
bun run dev
```

## Building

Create a production build:

```sh
bun run build
```

Preview the production build:

```sh
bun run preview
```

## Project Structure

- `src/routes/` - Pages and blog posts
- `src/routes/blog/` - Blog posts written in MDsveX (.svx files)
- `src/lib/components/` - Reusable Svelte components
